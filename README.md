# College Event Management System

## 📌 Description
A full-stack web application to manage college events, students, registrations, payments, and attendance.

## 🚀 Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js, Express.js
- Database: MySQL

## ✨ Features
- Event creation and management
- Student registration system
- Payment tracking
- Attendance management
- Admin dashboard

## 🛠️ How to Run
1. Clone repository:
   git clone https://github.com/mandlikavnish/Event-management-system

2. Install dependencies:
   npm install

3. Start server:
   node app.js
